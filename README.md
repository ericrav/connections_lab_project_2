## Set up

```sh
npm install
npm run dev
```

Open project at http://localhost:3000

## Libraries Used

- [face-api](https://github.com/justadudewhohacks/face-api.js)
- [Tone](https://tonejs.github.io/)
- [Express](https://expressjs.com/)

### Build Tools

- [Vite](https://vitejs.dev/)
